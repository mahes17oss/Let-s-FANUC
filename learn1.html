G-Code

CNC machines work by following the commands or instructions (G-codes / M-codes) which are given in Part Program. Here are G-codes for Fanuc cnc control which are necessary for a cnc machinists to learn to understand cnc programming.

# G-Code Handle
1 G00 Rapid traverse
2 G01 Linear interpolation
3 G02 Circular interpolation CW
1 G03 Circular interpolation CCW
2 G04 Dwell
3 G09 Exact stop
1 G10 Programmable data input
2 G20 Input in inch
3 G21 Input in mm
1 G22 Stored stroke check function on
2 G23 Stored stroke check function off
3 G27 Reference position return check
1 G28 Return to reference position
2 G32 Thread cutting
3 G40 Tool nose radius compensation cancel
1 G41 Tool nose radius compensation left
2 G42 Tool nose radius compensation right
3 G70 Finish machining cycle
2 G71 Turning cycle
3 G72 Facing cycle
1 G73 Pattern repeating cycle
2 G74 Peck drilling cycle
3 G75 Grooving cycle
2 G76 Threading cycle
3 G92 Coordinate system setting or max. spindle speed setting
1 G94 Feed Per Minute
2 G95 Feed Per Revolution
3 G96 Constant surface speed control
3 G97 Constant surface speed control cancel
